## 📌 **About SVM**

**Support Vector Machine (SVM)** is a supervised machine learning algorithm used for:

* 🔹 **Classification**
* 🔹 **Regression**

It works by finding the **best possible boundary** that separates different classes in the dataset.

Instead of using all data points, SVM focuses only on the **most important points (support vectors)** to build a robust model.
